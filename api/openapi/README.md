- `/api/v1/auth`
- `/api/v1/profile`
- `/api/v1/friends`
- `/api/v1/notifications`
- `/api/v1/chat`

The first implemented slice covers:
- signup
- login
- profile read/update
- friend requests and notifications
- chat conversation create/list
